The diy tools, herd design (from 5.99-B)

Includes litmus, herd, generators (diy,diyone,diycross) and some tools

This is a snapshot of the development version.

To compile and install in your home directory (binaries -> $HOME/bin,
library files -> $HOME/lib), just type "make luc" at the root of
the source tree. Then, provided you have $HOME/bin in your path,
all diy commands (such as litmus, diy etc.) can be invoked.

Requirements:
* Ocaml, version at least 4.00
  (Ocaml is available from http://caml.inria.fr/ocaml/)

* Ghostscript, for the GUI components to work
